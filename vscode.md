# 安装

下载地址 cdn 加速 http://vscode.cdn.azure.cn/

# 卸载
1. 控制面板卸载vscode
2. Win+r 输入%appdata%,删除 code 文件夹
3. 用户文件夹删除.code 文件夹

# 操作说明
**删除重复行** 打开命令台输入 lines，找到=delete duplicate lines=
**删除含有某些关键字的行** `.*(word1|word2|word3).*`
**自动折行** 设置搜索 word wrap 或者快捷键 Alt+Z
